# IT How-To Guides

This repository contains simple IT support how-to guides for common help desk and system administration tasks. Each guide is written in a clear step-by-step format and includes the purpose, audience, requirements, screenshots, expected results, verification steps, common issues, and documentation notes.

## Purpose

The purpose of this project is to practice writing clear technical documentation for IT support environments. These guides are designed to be easy for help desk technicians to follow and update.
